De Cero a Expertos:Tecnologías Aprendidas en el Curso de Desarrollo Front End

HTML

HTML (HyperText Markup Language) es el lenguaje que utilizo para estructurar contenido en la web. A través de etiquetas, organizo texto, imágenes, enlaces y otros elementos para crear la estructura básica de una página web.

CSS

CSS (Cascading Style Sheets) es lo que utilizo para darle estilo y diseño a las páginas web. Me permite controlar el aspecto visual de los elementos HTML, como colores, tipografías, tamaños, y márgenes, entre otros, haciendo que la página sea más atractiva y fácil de usar.

Javascrip

JavaScript es el lenguaje que uso para añadir interactividad a las páginas web. Me permite crear funcionalidades dinámicas, como animaciones, validación de formularios, o responder a eventos del usuario, haciendo que la página sea más interactiva y responsiva.

Vue.js

Vue.js es un framework de JavaScript que utilizo para construir interfaces de usuario dinámicas y reactivas. Me permite organizar el código en componentes reutilizables, facilitando el desarrollo de aplicaciones web más eficientes y escalables.
